# Linux For Beginners

Linux For Beginners is meant to run as a 3-week workshop, with one 2.5 hour session per week, in person.  The timing is spaced out to give people time to try things with their Linux systems and practice in the days in between sessions.

Feel free to pick up and use this curriculum in your own project, it's licensed [CC-BY](https://creativecommons.org/licenses/by/4.0/).

## Curriculum Is In Flux!

Please be advised that this will change often, as I just built it and am going through its first use.

## Other Materials:

To run a successful Linux For Beginners workshop, you will need a few things in addition to this curriculum:

- A general working knowledge of Linux on the desktop.
- Enough USB install disks for your attendees.  I'm using Linux Mint, so you may have to update some things if you've chosen another distro.
- Some refcards (reference cards) or cheat sheets.  I'm not hosting them here, as I haven't created my own, but I recommend a good cheat sheet for basic Bash commands at least.
- Snacks.  You're keeping people in session for 2.5 hours, and there may be questions after each session ends.  Students are easier to work with if they aren't hangry.
